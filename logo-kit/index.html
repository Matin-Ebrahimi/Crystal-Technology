07 — Spacing & Size

فضای امن و حداقل اندازه

فضای امن (Clear Space)

همیشه فضای خالی حداقل به اندازه‌ی ارتفاع نماد «M» را دور تا دور لوگو نگه دارید. هیچ متن، خط یا عنصر دیگری وارد این فضا نشود.

حداقل اندازه

برای خوانا ماندن، لوگوی افقی را کوچک‌تر از ۱۲۰ پیکسل عرض (یا ۳۲ میلی‌متر در چاپ) استفاده نکنید. زیر این اندازه فقط از نماد استفاده کنید.

08 — Usage Rules

باید و نباید

✓ درست

  • ✓ روی زمینه‌ی تیره از نسخه‌ی dark و روی زمینه‌ی روشن از نسخه‌ی light استفاده کنید.
  • ✓ فضای امن دور لوگو را همیشه حفظ کنید.
  • ✓ رنگ تیل دقیق (#22D3EE) را حفظ کنید.
  • ✓ برای فضاهای کوچک از نماد تنها استفاده کنید.
  • ✓ نسبت ابعاد را همیشه قفل نگه دارید.

✕ نادرست

  • ✕ لوگو را کج، کشیده یا فشرده نکنید.
  • ✕ رنگ لوگو را به دلخواه (مثلاً آبی قدیمی) تغییر ندهید.
  • ✕ نسخه‌ی تیره را روی زمینه‌ی روشن نگذارید (متن ناپدید می‌شود).
  • ✕ سایه، طرح یا افکت اضافه به لوگو نزنید.
  • ✕ فونت یا فاصله‌ی حروف را دستکاری نکنید.
